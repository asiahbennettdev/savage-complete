# savage-auth
Objective was to take starter code and manipulate showing the user name on the profile page, while user message is displayed. 

![21 Savage](public/prosav.png)

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:8383`
